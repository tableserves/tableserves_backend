## 🔧 Troubleshooting

### Not Receiving Notifications?

**Check 1: Browser Permission**
- Chrome: Settings → Privacy → Site Settings → Notifications
- Firefox: Settings → Privacy → Permissions → Notifications
- Safari: Preferences → Websites → Notifications

**Check 2: Connection Status**
- Look for green dot on notification bell
- Green = Connected ✅
- Red = Disconnected ❌

**Check 3: User Role**
- Must be logged in as:
  - Restaurant Owner, OR
  - Zone Shop Owner/Vendor

### Desktop Notifications Not Showing?

**Windows:**
1. Settings → System → Notifications
2. Make sure browser is allowed

**Mac:**
1. System Preferences → Notifications
2. Find your browser and enable

**Linux:**
1. Check notification daemon is running
2. Browser should have notification permission

### Sound Not Playing?

1. **Check sound toggle** in notification dropdown
2. **Check browser volume** (not muted)
3. **Check system volume**
4. **Try clicking anywhere** on page first (browser security)
